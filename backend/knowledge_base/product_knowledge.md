# PRODUCT CATALOG: Treelight Innovations

## 1. Flagship Product: Xact-Scope
- **Primary Function**: Automates the conversion of unstructured voice notes into structured Excel spreadsheets (.xlsx).
- **Industry Vertical**: Specifically engineered for Insurance Property Claims and Xactimate estimation workflows.
- **Key Technology**: Utilizes speech-to-text processing paired with structured data mapping to eliminate manual administrative overhead for adjusters.
- **Market Impact**: Reduces document preparation time by up to 80%, allowing for faster claim settlements.

## 2. Core Infrastructure: Cortex-RAG
- **System Type**: High-performance Retrieval-Augmented Generation (RAG) engine.
- **Data Sovereignty**: Features multi-user data isolation and persistent, topic-based memory stored in local vector databases.
- **Capabilities**: Supports codebase ingestion from GitHub and document-to-embedding conversion for instant "subject matter expertise".
- **Security**: Designed for enterprise-grade deployment using Docker to ensure data stays private.

## 3. Business Solutions: Enchanted Quill Framework
- **Project Type**: Full-stack Enterprise Management System.
- **Technical Stack**: Built with a React frontend and Python/FastAPI backend.
- **Operational Features**: Live inventory search, global payment processing, and automated sales report generation.
- **Scalability**: This architecture serves as the blueprint for custom E-commerce and Inventory management solutions.

# TECHNICAL SPECIFICATIONS (FOR AGENT REFERENCE)
- **Deployment**: All Treelight products are containerized via Docker for environment stability.
- **Architecture**: Microservice-oriented using FastAPI to ensure low latency and high scalability.

# THE CORE: Treelight AI-Core Architecture
- **Model Intelligence**: Powered by Gemini 1.5 Flash via FastAPI, providing high-speed, 128k context-aware reasoning.
- **Memory Engine (RAG)**: Utilizes a Dockerized ChromaDB instance for Retrieval-Augmented Generation.
- **Data Persistence**: Uses a local SQLite-based vector store to ensure user-specific data is never "forgotten" and never shared with public models.
- **Web Interface**: Deployed as a Next.js application, utilizing Tailwind CSS for professional, responsive design.
- **Security Protocols**: Features Clerk-based authentication and environment-isolated Docker containers for maximum client privacy.
